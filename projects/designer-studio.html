CASE STUDY

Project Overview

The Design Studio Management System is a full-stack web application developed to simplify the daily workflow of a design studio. It combines client management, project tracking, pricing, and online payments in a single platform.

Clients can register, submit design requests, monitor project progress, review completed work, and securely pay for services. Designers manage requests, update project statuses, and define service pricing through an intuitive administration panel.

The application also features a modern landing page with an interactive parallax effect, providing an engaging first impression while maintaining excellent usability across desktop and mobile devices.
